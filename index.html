T-Shirt Wahl

Diese Seite dient zur Wahl der T-Shirts bzw. des Mottos für den letzten Schultag.

Wenn der QR-Code richtig gescannt wurde, sollte in dem unteren Feld ein einzigartiger Code stehen. Dieser Code berechtigt zu jeweils einer Stimme.

Sobald eine Stimme abgegeben wurde, kann sie nicht mehr abgeändert werden.

  • Zur Auswahl stehen:

  • "ES": Bild
  • "Wer sich erinnert, war nicht dabei": Bild
  • "Abicetamol": Bild
  • "Abi con carne: Scharf im Abgang": kein Bild vorhanden
  • "ACAbi: kein Bild vorhanden"
